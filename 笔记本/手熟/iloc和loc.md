任务 1：精准筛选“暴利”品牌（练习 loc 的条件筛选）
背景： 你的研究发现联想内存溢价高达 2.33 倍，被判定为“暴利”。
练习题： 从“品牌硬件溢价对比分析.csv”中，使用 loc 筛选出所有评价为“暴利”且硬件项目为“内存”的行。
df.loc[(df['评价'] == '暴利')& (df['硬件类型'] == '内存条')]

任务 2：实现“48G 物理门槛”打标（练习 loc 的赋值能力）
背景： 决策树模型证明，内存容量是区分移动工作站（均价 3.95W）与主流产品的核心标。
练习题： 在“电脑数据.docx”生成的 DataFrame 中，使用 loc 创建一个新列 Tier_Level。逻辑是：如果“内存(G)” > 48，则赋值为“顶级旗舰”，否则为“常规机型”。
df.loc[df['内存(G)'] > 48 , 'Tier_Level'] = '顶级旗舰'
df.loc[df['内存(G)'] <= 48 , 'Tier_Level'] = '常规机型'

任务 3：数据切片与预览（练习 iloc 的位置定位）
背景： 面对 767 条深度评论 或 517 条零售价格数据，你有时只想快速看前几列或前几行。
练习题： 使用 iloc 提取“内存条.docx”数据表中的前 10 行和前 3 列（品牌、容量、代际）进行快速核对。
df.iloc[0:10, 0:3]

SELECT *
FROM df
WHERE 评价 = '暴利' AND 硬件类型 = '内存条';

SELECT *
    CASE
        WHEN '内存(G)' > 48 THEN '顶级旗舰'
        ELSE '常规机型'
    END AS 'Tier_Level'
FROM df;

SELECT 品牌、容量、代际
FROM df
