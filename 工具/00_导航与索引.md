## ⚡ 最常用的 3 个操作

### 操作 1：快速查询某行某列

```python
# 找所有"价格 > 5000"的行
result = df.loc[df['价格'] > 5000]

# 找"品牌=Apple"且"价格>3000"的行
result = df.loc[(df['品牌'] == 'Apple') & (df['价格'] > 3000)]

# 查前 2 行，前 3 列
quick_view = df.iloc[0:2, 0:3]
```

➡️ 详见：01_数据探索与清洁.ipynb § 1️⃣

---

### 操作 2：两表连接

```python
# 以 df_A 为主，从 df_B 抄价格（最常用）
df_merged = pd.merge(df_A, df_B, on='商品ID', how='left')

# 只要双方都有的
df_inner = pd.merge(df_A, df_B, on='商品ID', how='inner')
```

➡️ 详见：02_数据融合与变形.ipynb § 1️⃣

---

### 操作 3：分组求和

```python
# 按"品牌"分组，求"销售额"总和
result = df.groupby('品牌')['销售额'].sum().reset_index()

# 按"品牌"和"季度"分组，求均价和总销量
result = df.groupby(['品牌', '季度']).agg({
    '价格': 'mean',
    '销量': 'sum'
}).reset_index()
```

➡️ 详见：03_统计分析与展示.ipynb § 1️⃣
